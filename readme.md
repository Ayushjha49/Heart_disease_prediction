# Heart Disease Detection Using Random Forest (From Scratch)

## Project Overview
This project implements **Random Forest** from scratch to predict heart disease based on patient data.

- Dataset: UCI Heart Disease Dataset (`data/heart.csv`)
- Features: age, sex, cp, trestbps, chol, fbs, restecg, thalach, exang, oldpeak, slope, ca, thal
- Target: `target` (1 = heart disease, 0 = no heart disease)
